Assignment Objective

The purpose of Lab 05 & 06 is to understand and implement different CSS layout techniques used in modern web development.
This lab focuses on practicing layout methods such as Display properties, Float, Positioning, Flexbox, Grid, and Responsive Design using HTML5 and CSS3.

Project Structure

The project is organized into three main sections:

1. styles/

This folder contains all CSS files.
Each file is dedicated to a specific layout concept.

base.css – Common base styles used across all pages

display.css – Styles related to display properties

float.css – Float-based layout styles

position.css – Positioning (relative, absolute, fixed, sticky)

flexbox.css – Flexbox layouts and alignment

grid.css – CSS Grid layouts

media-queries.css – Media query rules

responsive.css – Responsive design enhancements

utilities.css – Reusable utility classes

2. exercises/

This folder contains practice exercises for each CSS layout topic.

display-exercises.html – Display property exercises

float-exercises.html – Float layout exercises

position-exercises.html – Position property exercises

flexbox-exercises.html – Flexbox practice

grid-exercises.html – Grid layout practice

media-query-exercises.html – Responsive and media query exercises

Each exercise file links only to the relevant CSS files from the styles folder.

3. layouts/

This folder contains real-world layout implementations using different CSS techniques.

magazine-float.html – Magazine-style layout using floats

holy-grail-comparison.html – Holy Grail layout comparison using different techniques

dashboard-grid.html – Dashboard layout using CSS Grid

portfolio-mixed.html – Portfolio layout using a mix of Flexbox and Grid

responsive-demo.html – Fully responsive layout demonstration

Main Entry File

index.html serves as the main entry point of the assignment.

It provides navigation links to all exercise pages and layout pages.
